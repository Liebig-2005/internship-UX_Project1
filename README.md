# Internship UX Project 1

## 📖 Project Explanation

This project was the first stage of my internship work, where I started applying **user experience (UX) design principles** in a real development environment. The main goal was to create a simple React application that demonstrates how design and functionality come together, while also learning how to plan and document the process.

At the beginning of Project 1, my focus was on **getting familiar with React** and understanding how to structure a front‑end project. I built the application using reusable components, which helped me learn how to organize code and keep the design consistent. Styling was done with CSS, where I experimented with layouts, colors, and spacing to make the interface visually appealing and easy to use.

Alongside the coding, I prepared an **estimation sheet** to plan out the project. This document included task breakdowns, timelines, and resource allocation. It was my first experience combining design and development with project management, and it gave me a better understanding of how professional projects are planned before they are built.

To showcase the design process, I included **screenshots** that represent the prototypes and final layouts. These visuals helped me explain the design decisions I made and how the interface evolved during development. They also served as a way to communicate the project to others without needing to run the code.

Overall, Project 1 was about **learning the basics** — both in terms of technical skills and UX thinking. It taught me how to start with an idea, plan it out, build it step by step, and present it clearly. While it was a simple project compared to Project 2, it laid the foundation for my internship journey and gave me the confidence to move forward with more complex and polished designs.

This project reflects my **first steps into combining design and development**, and it shows how I began to think about users, usability, and planning in a structured way.
